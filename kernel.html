Operating System Kernel Development

This project involves the design and implementation of a custom operating system kernel. The kernel is built in C and Assembly, featuring basic file system support, memory management, and real-time task scheduling. The project emphasizes understanding of low-level system architecture and how the kernel interacts with hardware. I am working to ensure that the kernel is lightweight and efficient, with the goal of creating a stable foundation for running applications.
