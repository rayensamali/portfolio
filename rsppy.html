Raspberry Pi IoT Project

This IoT project uses a Raspberry Pi to interface with various sensors such as temperature, humidity, and motion sensors. The readings from these sensors are transmitted to a cloud server using MQTT, enabling real-time monitoring and data collection. The project aims to showcase the capabilities of Raspberry Pi in the IoT ecosystem and its potential in smart home applications, environmental monitoring, and automation.
